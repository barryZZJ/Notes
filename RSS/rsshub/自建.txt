cache配置：
CACHE_TYPE: 缓存类型，可为 memory 和 redis，设为空可以禁止缓存，默认为 memory
CACHE_EXPIRE: 路由缓存过期时间，单位为秒，默认 5 * 60
CACHE_CONTENT_EXPIRE: 内容缓存过期时间，每次访问会重新计算过期时间，单位为秒，默认 1 * 60 * 60
REDIS_URL: Redis 连接地址（redis 缓存类型时有效），默认为 redis://localhost:6379/
MEMORY_MAX: 最大缓存数量（memory 缓存类型时有效），默认 256

debug:
json:
创建自定义对象。
将对象赋值给 ctx.state.json。
访问相应的路由 + .debug.json 来查看您的对象。
示例：
const info = utils.fetchAllCharacters(res.data, base);
ctx.state.json = {
	info,
	key: 'str',
};

debug.html
为了快速测试 ctx.state.data 中的 description 是否正确，你可以利用 debug.html 机制来获取相关条目的 HTML，该链接可以直接在浏览器中打开以预览渲染结果。
使用方式：访问相应的路由 + .{index}.debug.html，其中 {index} 为你的 ctx.state.data.item 中的项目序号（从 0 开始），即返回对应路由结果中的 ctx.state.data.item[index].description 信息。

图片反代
方便国内不挂VPN就查看图片；方便查看微博等具有反盗链的图片（只要没有referer就可以查看，但是有些阅读器会自动添加referer）
【两种方法不能同时使用！】
1. 环境变量配置
https://docs.rsshub.app/install/#pei-zhi-tu-pian-chu-li
''HOTLINK_TEMPLATE=https://i3.wp.com/${host}${pathname}''
''HOTLINK_INCLUDE_PATHS=/weibo,/picuki,/twitter''
''HOTLINK_EXCLUDE_PATHS=''

docker-compose作为environment设置时,由于${xxxx}默认会被docker-compse解析,需要配置两个$$,具体如下:
	environment:
		HOTLINK_TEMPLATE: 'https://images.weserv.nl/url=$${href_ue}'

2. 添加到通用参数配置
https://docs.rsshub.app/parameter.html#duo-mei-ti-chu-li
用法：需要环境变量ALLOW_USER_HOTLINK_TEMPLATE=true，在订阅url后添加通用参数
?image_hotlink_template=https://images.weserv.nl?url=${href}
?image_hotlink_template=https://i3.wp.com/${host}${pathname}
