====== classpath ======
Created 星期二 05 九月 2023

各种resource文件应放入src/main/resources，在pom中注册后，会即时自动拷贝到target/classes中

在pom中添加资源文件夹：
<build>
  <resources>
	<resource>
	  <directory>src/main/python</directory>
	  <includes>
		<include>**/*.py</include>
	  </includes>
	</resource>
  </resources>
  <!-- ... -->
</build>

路径：
"/"开头 对应"/target/classes"
相对路径 对应当前.class文件在target中的路径

https://www.geeksforgeeks.org/loading-resources-from-classpath-in-java-with-example/
getResource()
''URL url = getClass().getResource(fileName); // 静态方法使用ClassName.class替换getClass()''
''//注意直接用url.getPath()开头会多一个/，导致无法获取正确路径，需要用Paths转换一下''
''String path = Paths.get(url.toURI()).toString();''

''File file = new File(url.toURI());''
''file.exists();''

getResourceAsStream
''InputStream instr = getClass().getResourceAsStream(fileName);''
