====== 端口转发设置 ======
Created 星期四 18 五月 2023

1. 端口转发只对WAN口的数据包生效，LAN口不进行端口转发
2. 如果想转发给自己，需要填内部ip（如192.168.50.1），或WAN口的外网ip（10....），不能填127.0.0.1

yacd: 9999 -> 192.168.50.1

LAN口端口转发，手动配置路由表：

如：192.168.116.239:445 -> 192.168.116.239:21445
10.26.70.14:445 -> 192.168.116.239:22445
# DNAT规则
iptables -t nat -A PREROUTING -d 192.168.116.239 -p tcp --dport 445 -j DNAT --to-destination 192.168.116.239:21445
# 源地址伪装
iptables -t nat -A POSTROUTING -p tcp -d 192.168.116.239 --dport 21445 -j MASQUERADE

# 删除规则（如需要）
iptables -t nat -D PREROUTING -d 192.168.50.250 -p tcp --dport 445 -j DNAT --to-destination 192.168.116.239:21445
iptables -t nat -D POSTROUTING -p tcp -d 192.168.116.239 --dport 21445 -j MASQUERADE

# 查看NAT规则
iptables -t nat -L -n -v
