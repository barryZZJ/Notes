frps服务端：（服务器防火墙记得开放对应端口）（不要开kcp，否则会用不了）
# 一键配置脚本：https://github.com/MvsCode/frps-onekey
mkdir frps
cd frps
wget https://raw.githubusercontent.com/MvsCode/frps-onekey/master/install-frps.sh
mv install-frps.sh frps.sh
chmod 700 ./frps.sh
sudo ./frps.sh install  {install|uninstall|update|config}
（记得修改配置的log_file为/usr/local/frps/frps.log)

配置文件：/usr/local/frps/frps.ini

/etc/init.d/frps {start|stop|restart|status|config|version}
如果报错ine 45: [: 2431: binary operator expected，修改/usr/bin/frps，把$PID全部替换为${PID}

手动配置：
1. 下载frps，放入/usr/bin ln -s ~/frps/frps /usr/bin
2. 写好配置文件（见下），放入/usr/local/frps/frps.ini: ln -s ~/frps/frps.ini /usr/local/frps/frps.ini
3. 注册为服务：
https://gofrp.org/docs/setup/systemd/
apt install systemd
sudo nano /etc/systemd/system/frps.service
[Unit]
# 服务名称，可自定义
Description = frp server
After = network.target syslog.target
Wants = network.target

[Service]
Type = simple
# 启动frps的命令，需修改为您的frps的安装路径
ExecStart = /usr/bin/frps -c /usr/local/frps/frps.ini

[Install]
WantedBy = multi-user.target


sudo systemctl daemon-reload
systemctl start frps
systemctl enable frps

更新：
替换服务器的 /usr/local/frps/frps和/usr/bin/frps，给+x权限，然后重启服务即可。frpc的proxy断开后会自动重连【与login_fail_exit无关】
yy服务器使用版本：linux_386

systemctl {start|stop|status|restart|...} frps
