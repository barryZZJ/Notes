linux安装：
1. https://github.com/fatedier/frp
2. frpc放入/usr/bin
3. 注册为服务：
apt install systemd
sudo nano /etc/systemd/system/frpc.service

[Unit]
Description=Frp Client
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
Restart=always    
RestartSec=10
#ExecStartPre=/bin/sleep 30    # Add a delay of 30 seconds before starting
ExecStart=/usr/bin/frpc -c /home/barryoffice/frp/frpc_yy.ini
WorkingDirectory=/home/barryoffice/frp

[Install]
WantedBy=multi-user.target

4. sudo systemctl daemon-reload
systemctl start frpc
systemctl enable frpc

加密：
如果公司内网防火墙对外网访问进行了流量识别与屏蔽，例如禁止了 ssh 协议等，通过设置 use_encryption = true，将 frpc 与 frps 之间的通信内容加密传输，将会有效防止流量被拦截。

开启加密后，数据包会变大。这里的加密是指 frpc 和 frps 之间的通信，一般是为了防止被防火墙或者网关过滤。

压缩：
如果传输的报文长度较长，通过设置 use_compression = true 对传输内容进行压缩，可以有效减小 frpc 与 frps 之间的网络流量，加快流量转发速度，但是会额外消耗一些 cpu 资源。

压缩针对的是每个包比较大的情况，如果都是小包，可能反而会有负面效果。
