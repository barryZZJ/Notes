系统架构:
arch

memory:
free -h
cat /proc/meminfo

gpu:
nvidia-smi #显卡信息及使用情况
watch -n 1 nvidia-smi #实时查看，1秒刷新1次
nvidia-smi -L #list all available NVIDIA devices
sudo fuser -v /dev/nvidia* #查找占用GPU资源的PID
sudo kill -9 ***(PID) #解除显存占用
lspci | grep -i vga #查看显卡信息
lspci | grep -i nvidia #查看nvidia显卡信息
lspci | egrep 'VGA|3D' #显示所有显卡信息
import torch
torch.cuda.is_available()
torch.cuda.device_count()
torch.cuda.get_device_name(id=0)  # GPU型号
torch.cuda.current_device()

cpu cores:
cat /proc/cpuinfo # 查看CPU信息
cat /proc/cpuinfo | grep processor | wc -l #核数

cpu load:
top

三个数值分别表示1分钟、5分钟、15分钟的系统平均负载情况：
top -n 1 -b | grep "load average:" | awk '{print $12 $13 $14}' 

disk:
df -h

ps -aux | grep python

cat /etc/os-release
cat /proc/version
