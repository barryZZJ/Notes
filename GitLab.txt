**Backup**:
https://docs.gitlab.com/ee/administration/backup_restore/backup_gitlab.html#command-line-interface
sudo apt-get install rsync
cli:
sudo gitlab-backup create #SKIP=tar
配置备份文件路径：config/gitlab.yml
默认备份至/var/opt/gitlab/backups,
文件名为 [TIMESTAMP]_gitlab_backup.tar
e.g.  1493107454_2018_04_25_10.6.4-ce_gitlab_backup.tar

manually backup configs:
https://docs.gitlab.com/omnibus/settings/backups.html#backup-and-restore-omnibus-gitlab-configuration
