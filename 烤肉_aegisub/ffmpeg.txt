分割视频：
''ffmpeg -ss 00:00:00 -t 00:00:30 -i test.mp4 -c copy output.mp4''
	-ss start time offset 可以是时分秒毫秒，也可以是秒数
	-sseof set the start time offset relative to EOF
	-t duration 可以是时分秒毫秒，也可以是秒数 (不设置则默认到结尾) 
	-to stop time
	-i video file
	-report 保存日志到txt

倍速播放
视频的PTS系数为 1/倍速
e.g. 2倍速：
ffmpeg -i input.mkv -filter_complex "[0:v]setpts=0.5*PTS[v];[0:a]atempo=2.0[a]" -map "[v]" -map "[a]" output.mkv

提取音频
ffmpeg -i input-video.avi -vn -acodec copy output-audio.aac
-vn is no video.
-acodec copy says use the same audio stream that's already in there. (can be 'aac' 'm4a')(or -c:a copy)
