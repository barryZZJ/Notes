裁剪视频：
''ffmpeg -ss 00:00:00 -t 00:00:30 -i test.mp4 -c copy output.mp4''
	-ss start time offset Format: [HH:]MM:SS[.m...]  or ss[.m...][s|ms|us] 可以是时分秒毫秒，也可以是秒数. '55' '0.2' '200ms'
	-sseof set the start time offset relative to EOF
	-t duration 可以是时分秒毫秒，也可以是秒数 (不设置则默认到结尾) 
	-to stop time
	-i video file (要写在-ss之后，可以提高效率)
	-report 保存日志到txt
	-c codec for both video and audio

倍速播放
视频的PTS系数为 1/倍速
e.g. 2倍速：
ffmpeg -i input.mkv -filter_complex "[0:v]setpts=0.5*PTS[v];[0:a]atempo=2.0[a]" -map "[v]" -map "[a]" output.mkv

提取视频
ffmpeg -i test.mp4 -c:v copy out.mp4

提取音频
ffmpeg -i test.mp4 -c:a copy -vn output.aac
ffmpeg -i input-video.avi -vn -acodec copy output-audio.aac
-vn is no video.
-acodec copy says use the same audio stream that's already in there. (can be 'aac')(or -c:a copy)

音视频封装：
ffmpeg -i video.mp4 -i audio.aac -c:v copy -c:a copy -map 0:v:0 -map 1:a:0 output.mp4
ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 output.mp4
