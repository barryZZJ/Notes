====== ts源裁剪导出流程 ======
Created 星期一 27 二月 2023

参考织梦字幕组压制 字幕提取流程及教程.pdf

=== MeGUI处理音频： ===
1. HD Streams Extractor 分离音频aac
2. File Indexer打开ts源，使用默认的DGIndex，然后点Queue，得到d2v文件（需要另外保存一份为utf8编码）（记录输出音频的delay）
3. AVS Script Creator打开d2v文件，点save，得到avs脚本，视频预览界面关掉
4. AVS Cutter，打开avs脚本，通过set设置要切的各个分段（记得剪掉提供部分的广告），然后点 do all and close，得到clt文件（或手动点击Save cuts file）
5. 回到主界面Audio encoding部分，载入HD音频文件，在Cuts部分载入刚才的clt文件，encoder选AAC，delay会自动填写，点Queue
6. 得到剪好的音频文件.m4a

=== VS处理视频： ===
打开tmp_d2v.vpy
修改source和clt_file，使用utf8编码的d2v文件，自动读取clt剪辑视频
可以考虑使用FadeIn, FadeOut滤镜实现转场（但是音频没法淡入淡出）
任务管理器详细信息里把优先级调高

== 反交错处理（1080i转1080p） ==
	用potplayer或DGIndex确认ts的scan type为interlacing，如果为progressive则不需要反交错。
	无损反交错处理 [[反交错]]
	有损反交错：core.znedi3.nnedi3

preset选择ax264 ts
--demuxer y4m --preset veryslow --ref 8 --merange 24 --me umh --bframes 10 --aq-mode 3 --aq-strength 0.7 --deblock 0:0 --trellis 2 --psy-rd 0.6:0.1 --crf 18.5 --output-depth 8 -o "H:\萌音出演\テレビ\CUT\vs.mp4" -
（preset是用时间换画质{veryslow, slower, slow, medium}，crf是用空间换画质，越小码率越高）
(veryslow 18.5 码率 < slow 18 √)
注意把焦点切换到别的窗口，否则不小心再次按到start就得重新压制了

=== 处理字幕 ===
1. ts拖入Caption2AssNorubi.bat，得到提取不带rubi的_norubi
2. _norubi拖入SubCleanerテレビ 对字幕做清理、合并、去时间重叠，得到_cleaned
3. cut_ass.bat设置offset参数为第一步中HD Streams Extractor预览中的delay或File Indexer取出的音频的delay（负数），拖入_cleaned.ass和.clt，按成片裁剪字幕，得到_cut

=== 测试效果 ===
PotPlayer打开视频，拖入音频+字幕

== FFMPEG ==
mergeTwoFiles.bat
{{{code: lang="sh" linenumbers="False"
@echo off
cd %~dp1

set count=0
for %%a in (%*) do set /a count+=1

if %count% neq 2 (
    echo need exactly two arguments!
    exit /b
) else (
    set video=%1%
    set audio=%2%
    if "%video:~-4%" neq ".mp4" (
        set video=%2%
        set audio=%1%
    )
    set output=%~naudio%_cut.mp4
    echo video: %video%
    echo audio: %audio%
    @REM pause
    echo ffmpeg -i %video% -i %audio% -c:v copy -c:a copy -map 0:v:0 -map 1:a:0 %output%
)

pause
}}}
