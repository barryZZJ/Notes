====== 共享文件夹 ======
Created 星期三 23 八月 2023

在网络共享中心启动公用文件夹共享，然后共享文件夹需放在非NTFS目录，即可不用密码访问

网络位置 是针对的是广域网！
是用来打开网站、FTP站点或者网络位置
网络驱动器是针对的是局域网！

445端口被禁：
法1：端口转发【listenaddress必须是可达的ip地址】【注意防火墙设置】【优先级是先访问本地445，如果不可达再根据portproxy设置进行转发。所以需要监听端口的445不可达】
【新建一个虚拟网卡】
【经过各种测试感觉貌似这个方法不管用，本地连接时没走转发】
https://blog.csdn.net/babytang008/article/details/120489133
服务端：
netsh interface portproxy add v4tov4 listenport=23345 listenaddress=0.0.0.0 connectport=445 connectaddress=127.0.0.1
客户端：客户端上把虚拟网卡的ip的445转发到服务端的非445（如23345）端口，然后映射网络驱动器填这个ip即可【不用关闭本地的smb】【注意服务器的防火墙开启445和23345端口】
netsh interface portproxy add v4tov4 listenport=445 listenaddress=lanshare.barryzzj.top connectport=23345 connectaddress=dorm.barryzzj.top
netsh interface portproxy add v4tov4 listenport=445 listenaddress=wanshare.barryzzj.top connectport=23345 connectaddress=yy.barryzzj.top
#netsh interface portproxy add v4tov4 listenport=445 listenaddress=192.168.20.2 connectport=23345 connectaddress=wanshare.barryzzj.top

netsh interface portproxy show all

Get-SmbServerConfiguration | Select EnableSMB2Protocol
Set-SmbServerConfiguration -EnableSMB2Protocol $false
