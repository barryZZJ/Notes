====== 创建midi谱 ======
Created 星期五 24 四月 2026

去掉开头多余部分:
1. Alt 滚轮放大鼓谱波形
2. 关闭snap，选择要断开的地方，右键 Split / 选择Edit模式，全选所有track，拖动开头标记
3. 选择move模式，所有track拖到开头
4. 打开tempotrack，调整bpm与歌曲对齐

添加midi鼓轨道：
1. track空白处右键 - Insert Instrument - sforzando
2. sforzando加载SM Drum kit
3. Alt 4 打开 Step Sequencer  |  双击轨道 / Alt 3 打开 Piano Roll View （不会调默认Velocity）
4. Pattern - Load Pattern - SM Drum Kit  | 右键琴键图标-Use these Note Names Instead - SM Drum kit

导出：
File - Save As - MIDI Format 0

调整默认note velocity：
Step sequencer: Options - Set Default Velocity for Steps
	调整单个note的Velocity：展开track的下箭头
Piano Roll：新note默认与上一次创建的note拥有相同的velocity

修改note时长：
在midi编辑窗口上面的不同音符

开关自由选择：
开关上面的Snap。并且可以设置最小单位。
会影响Selection、添加midi note等

快捷键：
ctrl 滚轮：左右移动
alt 滚轮：缩放
views - virtual controllers
Ctrl F3: 播放时打开节拍器
Alt Shift 5: tempo track
G: go to
L: loop 开启循环播放， Shift L：循环播放当前选择部分
选择多个track：按住ctrl 点击轨道前的序号
Ctrl 1~3 : Snap to 1/4, 1/8, 1/16 note
