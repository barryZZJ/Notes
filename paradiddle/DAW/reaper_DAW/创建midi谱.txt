====== 创建midi谱 ======
Created 星期五 24 四月 2026

调整BPM：
右下角，先选择：Project timebase: Time，然后调整BPM

新建工程

导入音频文件：
1. 分别导入原版音频、drums、no drums
2. 选择三个轨道 - 按G (Group items） 这样可以同步移动
3. 确认bpm（记为t0）：修改工程bpm，往后拖至音乐的鼓点与小节起始（n.1）竖线对齐，检查bpm是否与鼓点对上。【注意：对齐的时候竖线提前一点，因为要敲击之后才会发声】
4. 点击音频开头，Alt S 开启吸附，拖选从音频开头往后至第一个长度大于1.5s的小节起始竖线，记下这个时长x秒，也可以记一下对应的结束位置（这个竖线将成为小节起始线，注意对齐实际节拍）。
	（拖选的部分应该没有鼓点。如果存在鼓点，则需要向后移动音频，使得第一个鼓点落在≥1.5s的位置，原来的起始位置作为音频开头和项目开头，此时x为项目开头到第一个鼓点的长度。此时视音频文件开头为项目开头，对应第6步不需要再移动。最后记得导出这个音频）【选择1.5s是因为游戏中节拍器第一拍的声音在1.5s响，为了保证第一拍的声音跟reaper中的能对应上】【经测试，修改bpm时，节拍器的声音还是尽量维持高低低低，只是间隔不一样了，所以没办法通过控制修改bpm的时b机来控制节拍器第一拍响的时机】
6. 计算60/x=t，记住原始bpm t0
7. 把音频拖至项目开头，并点击项目开头，Shift+C，Set tempo填入t，Set time signature选**1**/4。（此时会发现刚才结束的位置正好落在第一个竖线）
8. 选择第一个竖线（2.1的），Shift+C，Set tempo填入实际bpm t0，Set time signature选**4**/4。（此时鼓点会与后续的竖线对齐）
9. 设置project显示的measure和时间
	- 点击音乐实际第一小节开始的位置，Alt Enter (File - Project Settings) - Change start time/measure - Set measure 1 to nearest measure ...
	- 点击音频文件开头的位置，Ctrl 1 添加一个marker，然后 Alt Enter (File - Project Settings) - Change start time/measure - Set 0:00 to current edit cursor
9. 开启Spectro peaks + Spectrogram (shift P)
	Peaks display settings (Ctrl Alt P) 加载预设后进行微调

添加midi鼓轨道：
1. 右键轨道空白部分-Insert virtual instrument on new track
	2. 选择sforzando VST3i
	3. 加载SM Drums kit
2. Insert - new MIDI item	
3. track 命名为DRUM expert，拖动开始位置与音频文件对齐，拖动结束位置直到末尾。右键 - Item Settings 关闭 Loop item source
4. 双击新增的轨道 打开note编辑界面
	加载CCnames：File - Note / CC Names - Load - SM Drum Kit
	开始听写鼓谱
7. 最后统一检查一下velocity
	hi-hat closed >20, <110
	hi-hat open > 110
	hi-hat foot < 20
	Ride >20, <110
	Ride-bell >110

切换note Velocity：自动复制上一个选中的note的velocity

调整播放速度：
1. 调整右下角playback rate旋钮
2. 或：同调整BPM，但选择Beats (auto-stretch at tempo changes)
调整track速度：
Alt + 左键 拖动开头或结尾

中途切换BPM：(Shift+C)
Insert - Tempo/Signature change marker

导出midi：
1. 把midi轨道取消静音，高亮midi轨道
2. 选择音频文件的头尾
3. File - Export project MIDI
	- Time selection only
	- Selected tracks only
	- type 1 MIDI file
	- 勾选 Embed project tempo
	（说明：如果选Entire Project，则默认起始时间(0:00)是整个项目最左边。没到项目开头的会在这一段添加空白。如果选Time selection only，则起始时间(0:00)是selection的开头）
6. ParaUtils的试听的bpm可能不对，检查输出rlrr文件的length就好。
7. 检查时间是否对齐：选中midi开头，File-Project settings-Project start time - Set 0:00 to current time edit cursor

导出音频：如果音频开头进行了调整，需要重新渲染音频文件。（使用已保存的preset: Paradiddle）
高亮音频轨道
File - Render - 
	Source: Selected tracks
	Bounds: Entire project
	Sample rate: 与源文件相同
	Format: OGG Vorbis
	VBR quality: 0.5以下 
- render

关闭自动循环：
右键MIDI track - Item Settings - Loop item source

自动滚动：
Options - Automatically scroll view during playback

切换频谱：
Options - Peaks display Mode
可以保存预设
*ctrl alt p: show peaks display settings
*Alt P: toggle spectrogram
*Shift P: toggle spectral peaks
*Alt Shift P: toggle spectral peaks + spectrogram

经验：
除了Drums，其他三个轨道都mute，根据情况Solo原曲和midi轨
修改难度时批量删除八分音符：Edit - Quantize - Quantize position to grid
开镲别的游戏一般是放到Ride17上，我们可以通过控制velocity实现

不同难度经验：
- Hard: 删除16分音符
- Medium: 删除8分音符
- Easy: Medium试打一遍，删除打不明白的部分

频谱经验:(Drum preset)
高|暗粉色底鼓、矮红棕色军鼓、矮|亮粉色嗵鼓（越红越高）
蓝色延长音为crash，越长、越碎/音调越低尺寸越大、
比较哗~的声音可能是两个镲一起打发出的

调整grid粒度：Alt+G开关grid
midi编辑窗口：下方菜单可调整grid粒度
主界面：Options - Snap/Grid (Alt + L) - Show grid. line spacing

搜索快捷键和操作：Actions - Action list

快捷键：
Alt S: 自动吸附
S: split
R: 循环播放
*shift 滚轮：左右平移（可以在编辑快捷键中修改滚轮方向）(scroll horizontally reversed)
*alt 滚轮：上下平移(scroll vertically)
pgup pgdown ctrl 滚轮：上下缩放轨道
*Shift+A: Insert note at edit cursor (advance by grid)
*A: Insert note w/ vel 100 at edit cursor (advance by grid)
*Ctrl + 数字：insert note
*数字：跳过长度
*Ctrl + 左键双击: Insert note w/ vel 100 at mouse cursor

*Up/*Down: Increase / Decrease pitch cursor one semitone

编辑快捷键：Actions - show action list
