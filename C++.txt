c++11 lambda表达式（匿名函数）：
完整格式：
[capture list] (param list) -> return type {function body}
		capture list：函数中需要用到的外部变量需要捕获
值捕获：[a]；引用捕获：[&a]；
自动推断值捕获：[=]；自动推断引用捕获：[&]。
	   参数部分可省
	   返回类型部分可省
[](int a, int b){ return a < b; }
