模板类型，用于overload，如果有多个参数都使用该模板类型，则要求他们类型一致：
T = TypeVar('T')  # 通用类型，用于类定义，继承自Generic[T]，字类需要指定T的类型。
T = TypeVar('T', bound=TypeName)  # 绑定T到TypeName，是固定的。不支持Union

T = TypeVar('T')  # Can be anything
S = TypeVar('S', bound=str)  # Can be any subtype of str
A = TypeVar('A', str, bytes)  # Must be exactly str or bytes

T = TypeVar('T', bound='类名')  # 绑定T到类名，是固定的。
成员函数(sth: Type[T]) -> T:...

在类中注解自身：
def f(arg: Type['ClassName']) -> "ClassName"


取备选值中的一个：
Literal["val1", "val2"]
此时传入参数必须是具体的字符串常量，而不能是变量。解决方法：@overload一个函数定义，参数类型指定为str
e.g.:
''def open(path: _PathType,''
		 ''mode: Literal["r", "w", "a", "x", "r+", "w+", "a+", "x+"],''
		 '') -> IO[Text]: ...''

''# Fallback overload for when the user isn't using literal types''
''@overload''
''def open(path: _PathType, mode: str) -> IO[Any]: ...''


TypedDict
类似pydantic的module的方式，只用于类型检查，运行时为dict类型.
''class Point2D(TypedDict):''
	''x: int''
	''y: int''
	''label: str''
''Point2D = TypedDict('Point2D', {'x': int, 'y': int, 'label': str})''
''Point2D = TypedDict('Point2D', x=int, y=int, label=str)''
''a: Point2D = {'x': 1, 'y': 2, 'label': 'good'}  # OK''
''b: Point2D = {'z': 3, 'label': 'bad'}           # Fails type check''

''assert Point2D(x=1, y=2, label='first') == dict(x=1, y=2, label='first')''

overload:
https://docs.python.org/3/library/typing.html#typing.overload
类方法可以定义多个@overload装饰的函数和一个不带的函数，用于IDE静态解析，而不是真的执行。
（另一个可以真的执行不同函数的来自from overloading import overload https://peps.python.org/pep-3124/ ）
''from typing import overload''

''class Duck:''

	''@overload''
	''def quack(self) -> None: ...''

	''@overload''
	''def quack(self, mark: str) -> None: ...''

	''# 以上两个方法最终会被这个方法覆盖掉''
	''def quack(self, arg=None):''
		''if arg:''
			''print(f"GaGaGa: {arg}")''
		''else:''
			''print("GaGaGa!")''
