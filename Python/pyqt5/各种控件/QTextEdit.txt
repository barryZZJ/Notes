====== QTextEdit ======
Created 星期四 10 十一月 2022

https://doc.qt.io/qt-6/qtextedit.html

用于显示大段HTML或md文档

内容：
.html
setHTML(str)  以html格式解析，展示解析后的内容
toHTML() -> htmlstr  把显示内容转换为html后返回
Supported HTML Subset
https://doc.qt.io/qt-6/richtext-html-subset.html

.markdown
setMarkdown(str)
toMarkdown() -> str

.plainText
setPlainText(str)  不解析，原封不动显示
toPlainText()  -> str  把内容原封不动返回

插入文本：
使用QTextCursor
或直接在当前指针位置插入：insertHtml(), insertPlainText(), append() or paste()

textCursor()
setTextCursor()
selectAll()
copy()
cut()
paste()
clear()

支持redo, undo, zoomIn, zoomOut

设置WordWrap：
setLineWrapMode
默认空格处断行，NoWrap, FixedPixelWidth, ...

find() find and select a given string

documentTitle() function will return the text from within HTML <title> tags.


QTextDocument provides an isModified() function which will return true if the text has been modified since it was either loaded or since the last call to setModified with false as argument. In addition it provides methods for undo and redo.
