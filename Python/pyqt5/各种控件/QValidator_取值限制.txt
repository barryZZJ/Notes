====== QValidator 取值限制 ======
Created 星期二 22 十月 2024

https://doc.qt.io/qt-5/qvalidator.html

QIntValidator
QDoubleValidator
QRegExpValidator

QValidator::Invalid	0	The string is clearly invalid.
QValidator::Intermediate	1	The string is a plausible intermediate value.
QValidator::Acceptable	2	The string is acceptable as a final result; i.e. it is valid.

.validate(input: str)

与 QLineEdit, QSpinBox and QComboBox 连用:
QLineEdit.setValidator(v)  设置Validator，只允许输入Intermediate或Acceptable的内容，否则无法键入
setValidator(None) 删除
.validator() -> QValidator
.hasAcceptableInput() -> bool 最终需要手动调用这个方法确保输入的是Acceptable
