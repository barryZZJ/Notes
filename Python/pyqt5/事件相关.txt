=== 自动连接槽函数： ===
由于setupUi里调用了connectSlotsByName，
定义on_<object name>_<signal name>(<signal params>)成员函数后，
可以自动连接槽函数到对应的组件，不需要手动连接。
格式：
from PyQt5.QtCore import pyqtSlot
@pyqtSlot(<paramTypes>)
def on_<object name>_<signal name>(self, <signal params>):
	...
@pyqtSlot(int)
def on_mainwiz_currentIdChanged(self, id):
	...
https://blog.csdn.net/jia666666/article/details/81775614
https://doc.qt.io/qt-5/qmetaobject.html#connectSlotsByName
https://doc.bccnsoft.com/docs/PyQt5/signals_slots.html

=== 手动连接槽函数： ===
signal用.connect方法连接到槽函数
一：
designer中不设置，在代码中写：（如果信号带多个参数，对应槽函数包含多个参数即可）
def connectSlots(ui: Ui_Form):
	ui.xxButton.clicked.connect(lambda ui: func(ui)) # 或实现函数闭包，.connect(fun(ui))中fun(ui)返回一个函数即可
xxButton.clicked.connect(funcname) def funcname(txt): ...
self.connect(self.man_btn, QtCore.SIGNAL('clicked()'), self.manual_slot)

二：
designer中选择设置信号/槽，把按钮连接到form上，在form那一栏里编辑，添加槽函数。
新建QDialog的子类，实现刚刚添加的槽函数
self.chk_modsw.stateChanged['int'].connect(Form.stateChanged) # 0:no, 1:half, 2:check

=== 自定义信号与槽函数： ===
from PyQt5.QtCore import pyqtSignal, pyqtBoundSignal, pyqtSlot, QObject
class MyObj(QObject):
	# 要定义成类的静态成员
	started = pyqtSignal()
	newSignal = pyqtSignal(<type>)  # 如果信号有参数，type里指明参数类型，如str, int  # type: pyqtBoundSignal
	def doSomething(self):
		....
		self.newSignal.emit(<param>) # 发射信号

连接到外部的槽函数
1. myobj.signalName.connect(slotFunc) # 槽函数捕捉信号即可
myobj.started.connect(self.onStart)
def onStart(self, str):
	...
2. Obj(signalName=slotFunc) # 初始化时直接传递槽函数也可
