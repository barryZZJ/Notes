推荐配置：
label：horizontal policy设为Fixed
LineEdit: 文本框的sizeHint()似乎会比较大，建议Horizontal Policy设为Maximum，然后用fixed的Spacer把它挤小
Form Layout：
https://blog.csdn.net/liang19890820/article/details/51539339
https://blog.csdn.net/Vichael_Chan/article/details/100138499
layoutFieldGrowthPolicy 设为 FieldsStayAtSizeHint 防止文本框过长

设置窗口置顶：
WindowModality - WindowModal只相对于parent置顶，ApplicationModal相对于全局置顶。

=== 配合designer使用样例： ===
x可以把自动生成的Ui类改写成继承自QWidget的类，但是designer的改动不好同步。
√可以新建一个类既继承QWidget又继承pyuic生成的Ui_Form类。
初始化时调用self.setupUi
/layouts: layoutXxx.ui, layoutXxx.py
/windows: wndXxx.py （主窗体得用MainWindow，不能用Frame，要不然控件会挤在一起，也可能是设置没调好）
{{{code: lang="python3" linenumbers="False"
import sys
from PyQt5.QtCore import Qt, pyqtSlot
from PyQt5.QtGui import QIcon, QCursor
from PyQt5.QtWidgets import QApplication, QDialog, QMainWindow, QMessageBox, QLineEdit
from layouts.layoutMain import Ui_MainWindow  # 由Designer+pyuic生成
from utils.gui.exception_hook import UncaughtHook, on_exception  # 见'exception hook'


class MainWindow(QMainWindow, Ui_MainWindow):
    def __init__(self):
        super().__init__()
        self.setupUi(self)
        self.retranslateUi(self)

        # window settings
        # self.setFixedSize(self.width(), self.height())
        self.setWindowFlags(Qt.WindowMinimizeButtonHint | Qt.WindowCloseButtonHint)
    
        # exception handling
        err_hook = UncaughtHook()
        err_hook._exception_caught.connect(lambda msg: on_exception(self, msg))
        
        # 其他初始化设置...
        # self.connectSlots()  # 手动连接槽函数
    
    
    """utils"""

    """slots"""
    # 自动连接槽函数
    @pyqtSlot()  # 如果有参数的话，装饰器pyqtSlot()里必须要加上参数类型
    def on_pushButton_clicked(self):
        # 命名格式on_Child_SignalName，主窗口只能手动连接槽函数
        print('button clicked!')

if __name__ == '__main__':
    app = QApplication(sys.argv)
    mainwindow = MainWindow()
    mainwindow.show()
    sys.exit(app.exec_())  # 如果有多线程，需要os._exit(app.exec_())

}}}
