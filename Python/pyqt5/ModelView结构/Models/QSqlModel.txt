====== QSqlModel ======
Created 星期四 08 十二月 2022

QSqlQueryModel 
https://doc.qt.io/qt-6/qsqlquerymodel.html
对数据库的只读操作

QSqlTableModel 
https://doc.qt.io/qt-6/qsqltablemodel.html
对数据库的读写操作
(db=QSqlDatabase) # db必须先open
.setTable('tablename')  # 注意表名不能含`.`，建议做一层mapping
.setEditStrategy
.setFilter 
.select() 更新查询结果
.setHeaderData(col, Qt.Horizontal, 'Dispname')  # 设置在view中的显示名字
修改数据：
1. 设置EditStrategy
2. 调用.setData()，会根据strategy确定何时修改数据库数据
3. 调用.submitAll() 如果策略为OnManualSubmit  （不需要调用.select()）

example 
https://doc.qt.io/qt-6/qtsql-tablemodel-example.html
https://doc.qt.io/qt-6/qtsql-cachedtable-example.html
{{{code: lang="python3" linenumbers="False"
db = QSqlDatabase.addDatabase('QSQLITE', pubtype.name)
db.setDatabaseName('cache.db')
db.open()
model = QSqlTableModel(parent, db)
model.setTable('tablename')
model.select()
model.setHeaderData(0, Qt.Horizontal, 'newheader')  # 修改显示的列名

view = QTableView()
view.setModel(model)
view.hideColumn(1)  # 隐藏对应列
view.resizeColumnsToContents()  # 自适应每一列的列宽
}}}

QSqlDatabase
https://doc.qt.io/qt-6/qsqldatabase.html
用于连接到数据库
do not keep a copy of the QSqlDatabase around as a member of a class
db = QSqlDatabase.addDatabase('QSQLITE', connname)
db.setDatabaseName('test.db')  # 数据库文件名 注意调用open后才会新建文件
.open() 执行后才会创建文件
.exec(query) -> QSqlQuery  https://doc.qt.io/qt-6/qsqlquery.html
	select结果
	返回生成器：
	while res.next(): # 如没有匹配则返回False
		yield res.value(0)  # 数字与select的内容顺序一致

.tables()
.record(tableName) -> QSqlRecord
.transaction()
.commit()
.rollback()
.lastError()  -> QSqlError https://doc.qt.io/qt-6/qsqlerror.html

继承方法：
{{{code: lang="python3" linenumbers="False"
def __init__(self):
    super().__init__(super().addDatabase('QSQLITE'))
    self.setDatabaseName(str(PATHS.DB))
}}}
