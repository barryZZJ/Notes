处理.yaml文件
https://pyyaml.org/wiki/PyYAMLDocumentation

结构示例，支持嵌套、支持列表、反斜杠不需要转义
Key:
  subkey: value
  subkey: 'value1 value2'
  subkey: "value1 value2"
  subkey:
	- listitem1
	- listitem2
  subkey: [listitem1, listitem2]

"key: " 返回空字符串
"key:
  -" 返回[None]

pip install pyyaml

import yaml
try:
	from yaml import CLoader as Loader, CDumper as Dumper
except ImportError:
	from yaml import Loader, Dumper

stream = open(xxx)
data = load(stream, Loader=Loader)
output = dump(data, Dumper=Dumper)

解决dump时中文变为\u编码问题：在dump中添加参数allow_unicode=True
