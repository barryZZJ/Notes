https://docs.pydantic.dev/latest/usage/models/

各种类型转换 https://docs.pydantic.dev/latest/usage/types/

动态定义模型
https://docs.pydantic.dev/latest/usage/models/#dynamic-model-creation

必须参数/可选参数：
https://docs.pydantic.dev/latest/usage/models/#required-fields

(for v2.0:
class Model(BaseModel):
	# required
	a: int
	c: int = Field(...)
	d: Optional[int]  # can be None
	# optional, can be None
	b: int = 1
	e: Optional[int] = None
	f: Optional[int] = 1
	
int类型默认Field(strict=False)，即可以传入'45'等
v2定义str不能传入int型，需要把变量类型改为int
https://docs.pydantic.dev/latest/usage/fields/#strict-mode

动态默认值：
https://docs.pydantic.dev/latest/usage/models/#field-with-dynamic-default-value

解析数据：
Model(**kwargs)
Model.parse_obj(dict) -> deprecated, use .model_validate
.parse_raw(str) -> deprecated, use .model_validate_json
.parse_file

保存：
不支持直接dump为json
model.dict() -> deprecated, use model_dump(mode='json', by_alias=True)  # 如果mode='python'，则有些值如URL, datetime可能直接是对象
参数：by_alias 默认为False，使用field name作为存储的key；为True时使用alias name作为存储的key
.json -> deprecated, use model_dump_json


pydantic.parse_obj_as(model, obj)
pydantic.parse_raw_as


Model signature:
alias:
https://docs.pydantic.dev/latest/usage/fields/#field-aliases
_id: int = Field(..., alias='id')
(pydantic prefers aliases over names，在解析时会使用alias的key)

custom init:
''    def __init__(self, id: int = 1, *, bar: str, **data) -> None:''
''        """My custom init!"""''
''        super().__init__(id=id, bar=bar, **data)''

const:
MY_CONST: str = Field('CONST', const=True)  # 此成员不能在内部函数引用，且外部不能通过类名引用
''__MY_CONST__'': str = 'CONST'  # 以魔法方式命名的变量名不会被pydantic解析
排除项：
Class variables which begin with an underscore and attributes annotated with typing.ClassVar will be automatically excluded from the model.

支持多级继承
''from pydantic import BaseModel''

''class BaseMessage(BaseModel):''
''    sender: str''
''    receiver: str''

''class TextMessage(BaseMessage):''
''    content: str''

''class ImageMessage(BaseMessage):''
''    url: str''
''    caption: str''

''text_msg = TextMessage(sender="Alice", receiver="Bob", content="Hello!")''
''image_msg = ImageMessage(sender="Alice", receiver="Bob", url="example.com/image.jpg", caption="Check this out!")''


{{{code: lang="python3" linenumbers="False"
class MyTeam(BaseModel):
	id: str = Field(alias="_id")
	admin_id: str
	icon: str = None
	member_ids: List[str]
	name: str
	regDate: str
	tag_id: str

MyTeam.parse_obj({...})  ->  忽略多余参数
MyTeam.parse_raw("{...}")
}}}


python >= 3.9
{{{code: lang="python3" linenumbers="False"
from datetime import datetime
from typing import Optional
from pydantic import BaseModel


class User(BaseModel):
    id: int
    name = 'John Doe'
    signup_ts: Optional[datetime] = None
    friends: list[int] = []


external_data = {
    'id': '123',
    'signup_ts': '2019-06-01 12:22',
    'friends': [1, 2, '3'],
}
user = User(**external_data)
print(user.id)
#> 123
print(repr(user.signup_ts))
#> datetime.datetime(2019, 6, 1, 12, 22)
print(user.friends)
#> [1, 2, 3]
print(user.dict())
"""
{
    'id': 123,
    'signup_ts': datetime.datetime(2019, 6, 1, 12, 22),
    'friends': [1, 2, 3],
    'name': 'John Doe',
}
"""
}}}
