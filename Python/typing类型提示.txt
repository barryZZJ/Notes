强制类型转换库：pydantic

type hint / annotation

from typing import List, Tuple, Optional, Union, Any, Callable, overload
# Callable[[int], str] is a function of (int) -> str.

指定参数：
def f(a: int) -> None :

或 （参见docstring.txt）
def f(a, **kwargs):
	'''
	:type a:int
	:Keyword Arguments:
		* **key1** (*int*, *bytes*) --
			Description...
	'''

指定变量：
a # type: List[str]
单独写a: int 

检测：
isinstance(o, t)

T = TypeVar('T', bound='类名')
成员函数(sth: Type[T]) -> T:...
