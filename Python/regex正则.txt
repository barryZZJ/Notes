*可以通过把想要获取的地方用括号括起来
search(pattern, string, flags=0) 检查字符串的任意位置，返回第一个匹配 Scan through string looking for a match to the pattern, returning a match object, or None if no match was found.

match(pattern, string, flags=0) 检查字符串开头（相当于pattern前加了^） Try to apply the pattern at the start of the string, returning a match object, or None if no match was found.

findall(pattern, str) 尝试匹配多次，返回List[str|Tuple]，数组每个元素是字符串(<=一个括号)或字符串元组(多个括号)，元组包含各个括号的内容
finditer(pattern, string, flags=0) 同上，但返回生成器

fullmatch(pattern, string, flags=0) 相当于pattern前后加上^和$。Try to apply the pattern to all of the string, returning a match object, or None if no match was found.

sub(pattern, repl, string, count=0, flags=0) repl:替换结果，string:搜索字符串。repl中可以包含'\1'表示第一个括号中的内容。

split(pattern, string, maxsplit=0, flags=0) 用 pattern 分开 string 。 如果在 pattern 中捕获到括号，那么所有的组里的文字也会包含在列表里。

如果不匹配返回None，否则
搜索结果为Match对象：
	[i] 获取第i个括号内的匹配结果（0为全部匹配内容）
	.group() 返回整个匹配内容
		有括号时：.group(0)获得全部匹配内容；.group(1)获得第一个括号的内容...；
	.groups() 返回元组，获得每个括号内容，.groups(arg)把元组中所有未匹配到的None替换为arg


Pattern：
pattern = re.compile(r"xxx")
pattern.search()/match()/...

pattern字符串前加r可以防止 转义字符\ 起作用
r"\t" -> 输出 "\t"
