====== pydivert抓包 ======
Created 星期一 06 一月 2025

import pydivert

调用winapi，需要管理员运行
with pydivert.WinDivert("inbound and udp.DstPort == 46000 and udp.PayloadLength > 500") as w:
	for packet in w:
		addr: str = packet.ipv4.src_addr  # format: string "xxx.xxx.xxx.xxx"
		payload: bytes = packet.payload
		w.send(packet)

是拦截不是监听，如果不w.send，包是不会发出的

与wireshark的优先级关系：
outbound出站包：先经过pydivert，再经过wireshark
inbound入站包：先经过wireshark，再经过pydivert
