继承str类：
>>> class C(str):
...     def __new__(cls, value, *args, **kw):
...         return str.__new__(cls, value)
... 
>>> c = C("hello world")

含其他属性：
>>> class C(str):
...     def __new__(cls, value, meta):
...         obj = str.__new__(cls, value)
...         obj.meta = meta
...         return obj
... 
>>> c = C("hello world", "meta")
>>> c
'hello world'
>>> c.meta
'meta'
