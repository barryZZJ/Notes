=== 注册路由： ===
https://flask.palletsprojects.com/en/latest/api/#url-route-registrations
route:
添加参数：<key> 或 <type:key> types: {string(default), int, float, path, uuid}
在key中定义的可以加入函数形参
默认只处理GET
@app.route('/login', methods=['GET', 'POST']
@app.get()
@app.post()
非装饰器用法：
https://flask.palletsprojects.com/en/latest/api/#flask.Flask.add_url_rule
app.add_url_rule('/', view_func=func, methods=['GET','POST'])

config: https://flask.palletsprojects.com/en/3.0.x/config/
app.config[key] = val

url_for(func_name)  生成指定函数名对应的url，名称参数作为URL路径（如果是route变量）或query string

request object 获取请求参数: https://flask.palletsprojects.com/en/latest/api/#flask.Request
request.method  # str 'GET' 'POST'
request.form -> dict POST参数
request.args -> dict GET参数
request.values -> dict 同时包含GET和POST参数
request.files[key]
request.data
request.json POST参数按json解析

=== 响应 ===
return '', 404  # (可选)第二个参数可以指定状态码
return jsonify(dict), 404
response = make_response()
return response

=== Cookie: ===
response.set_cookie(key, val)

=== Session ===
from flask import session
app.secret_key = 'your_secret_key'
set: session[key] = val
get: session[key]
前端会多一个加密了的cookie: {'session': singed_encrypted_string}

===== 启动： =====
1. 直接运行py文件
2. flask --app pyname run -p PORT -h 0.0.0.0  文件名为app.py时不用加--app参数
其他参数:
--debug

使用flask启动时，__name__ == pyname，所以不会触发__name__ == '__main__'，把要想执行的代码放到全局中

abort(404) raise一个HTTPException，后面的代码不会运行 （服务器不会停止

=== 返回json ===
return jsonify({'result': result})
js解析：response.data.result

=== blueprints ===
方便路径模块化
所有bp最后全部在app.py中import，然后app.register_blueprint
https://flask.palletsprojects.com/en/latest/tutorial/views/

获取app对象：
from flask import current_app


===== WSGI HTTP Server: =====
部署环境，支持大规模并发请求。不用的话只能支持少量并发请求。
===== Linux: Unicorn =====
需要使用gunicorn运行：
pip install gunicorn
gunicorn pyfilename:appname -b 0.0.0.0:12345

===== Windows/Linux: waitress =====
https://docs.pylonsproject.org/projects/waitress/en/latest/arguments.html#arguments
pip install waitress
waitress.serve(app, host=, port=)

默认不会显示flask的web流量log，解决：
pip install Paste
from paste.translogger import TransLogger
waitress.serve(TransLogger(app), ...)
