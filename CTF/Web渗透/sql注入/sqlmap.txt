====== sqlmap ======
Created 星期五 22 十一月 2024

usage: https://github.com/sqlmapproject/sqlmap/wiki/Usage

python sqlmap.py -u URL?id=1 --batch
--dbs  获取所有数据库
--tables -D db_name  获取指定数据库的所有表
--columns -T table_name 获取指定表的所有列
-C col_name

--threads
