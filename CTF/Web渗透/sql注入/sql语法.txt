====== sql语法 ======
Created 星期一 25 十一月 2024

sql 语法：https://dev.mysql.com/doc/refman/8.4/en/sql-statements.html
单行注释：通用：-- 注意--后面要有一个空格，或者--+ --\t --\n等控制字符也可以。
mysql特有的是#
在url构造中，#编码为%23，空格编码为+，因此最好在url中用--+
多行注释：/* ... */
SELECT: https://dev.mysql.com/doc/refman/8.4/en/select.html
	LIMIT [offset,] row_count  其中offset从0开始
	LIMIT row_count OFFSET offset

逻辑运算
AND, &&	Logical AND
NOT, !	Negates value
OR, ||	Logical OR
XOR	Logical XOR
