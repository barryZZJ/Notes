====== postMessage父子窗体通信 ======
Created 星期三 01 三月 2023

由于子窗体iframe无法直接访问父窗体的document，因此需要通过postMessage进行通信

https://developer.mozilla.org/zh-CN/docs/Web/API/Window/postMessage
支持跨域通信

otherWindow.postMessage(message, targetOrigin, [transfer]);

消息必须是string，考虑使用JSON.stringify(), JSON.parse()

targetOrigin
通过窗口的 origin 属性来指定哪些窗口能接收到消息事件，其值可以是字符串"*"（表示无限制）或者一个 URI。在发送消息的时候，如果目标窗口的协议、主机地址或端口这三者的任意一项不匹配 targetOrigin 提供的值，那么消息就不会被发送；只有三者完全匹配，消息才会被发送。
如果你明确的知道消息应该发送到哪个窗口，那么请始终提供一个有确切值的 targetOrigin，而不是 *。不提供确切的目标将导致数据泄露到任何对数据感兴趣的恶意站点。

可以让iframe加载完成后向top发送ready的信息，然后top再postMessage，就可以解决同步问题了。
window.parent.postMessage()

URL A:
{{{code: lang="js" linenumbers="False"
// Get a reference to the iframe in URL B
const iframe = document.querySelector('iframe[src="https://iframe.org"]');

// Send a message to the iframe containing the text you need to pass
iframe.contentWindow.postMessage('text to pass', 'https://iframe.org');

// Listen for messages from the iframe
window.addEventListener('message', event => {
  // Check if the message came from the iframe
  if (event.source === iframe.contentWindow) {
    // The message contains the text we need
    const textFromIframe = event.data;
    // Do something with the text...
  }
});
}}}


In your code for URL B, you can listen for the message using the window.addEventListener method:
{{{code: lang="js" linenumbers="False"
window.addEventListener('message', event => {
  // Check if the message came from the parent window with the URL A origin
  let origin = event.origin || event.originalEvent.origin;
  if (origin === "https://example.org") {
    // The message contains the text we need
    const textFromA = event.data;
    // Do something with the text...
  }
}, false);

// Send a message to the parent window
window.parent.postMessage('text to pass', 'https://example.com');
}}}
