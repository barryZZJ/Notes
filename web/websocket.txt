====== websocket ======
Created 星期二 30 六月 2026

https://developer.mozilla.org/zh-CN/docs/Web/API/WebSockets_API

分为客户端和服务器

WebSocket 是一种网络通信协议，支持在单一 TCP 连接上进行全双工通信。它打破了传统 HTTP “请求-响应”的单向模式，允许服务器主动向客户端推送数据，从而实现毫秒级的实时交互。
核心优势
双向通信：客户端和服务端都能随时向对方发送消息。
低延迟：握手阶段通过 HTTP 完成，握手成功后升级为专有 TCP 连接。头部开销极小，数据传输高效。
持久连接：一次握手后保持长连接，避免了频繁创建 TCP 连接的性能损耗。
常见应用场景
实时互动：在线聊天室、多人游戏。
数据推送：股票行情、实况天气、传感器数据监控。
协同工具：实时文档编辑、AI 流式内容（Token Streaming）生成。
协议与测试协议标识符：
非加密使用 ws://（默认端口 80），加密使用 wss://（默认端口 443）。
调试工具：开发阶段可使用 WebSocket 在线测试工具 验证服务端接口的连通性。
