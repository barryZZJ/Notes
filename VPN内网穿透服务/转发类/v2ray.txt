====== v2ray ======
Created 星期四 30 三月 2023

https://guide.v2fly.org/prep/install.html

linux安装：
curl -O https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh
systemctl enable v2ray; systemctl start v2ray

更新：
cd ~/v2ray
bash install-release.sh

导出配置文件：
v2rayN客户端选中一个节点，右键-导出所选服务器为客户端配置，保存到json
需要编辑一下，outbounds里只保留tag为proxy的那一项，并且删掉routing字典

手动运行：
	v2ray run -c ~/v2ray/configs/jp4Direct.json
	yy服务器配置文件：~/v2ray/configs
可以使用 v2ray help 查看具体命令。

系统服务运行：
	systemctl {start|stop|restart|status} v2ray
	配置文件位于 /usr/local/etc/v2ray/config.json (建立软链接到~/v2ray/sysconf)

如果出现 Connection reset by peer ，需要把服务器的防火墙设为允许机场的ip
