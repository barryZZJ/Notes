====== nvidia cuda支持 ======
Created 星期三 07 八月 2024

windows：需要额外装wsl
https://docs.docker.com/desktop/gpu/

linux：
宿主机需要装cuda驱动
只需要在宿主机上装一个toolkit，改造一下docker客户端，就能让docker支持gpu，不需要在封装时做任何操作。

Nvidia Container Toolkit
install: https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/latest/install-guide.html
doc: https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/latest/index.html

Cuda:
https://developer.nvidia.com/cuda-downloads

Cuda on wsl

users must not install any NVIDIA GPU Linux driver within WSL 2.

Windows: https://docs.nvidia.com/cuda/wsl-user-guide/index.html#getting-started-with-cuda-on-wsl-2
1. 安装wsl 2
2. 在wsl中安装cuda for wsl-ubuntu https://developer.nvidia.com/cuda-downloads?target_os=Linux&target_arch=x86_64&Distribution=WSL-Ubuntu&target_version=2.0&target_type=deb_network
