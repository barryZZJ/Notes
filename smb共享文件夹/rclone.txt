====== rclone ======
Created 星期六 26 八月 2023

https://rclone.org/docs/
需要安装winfsp
rclone mount test:nspshare Z: --config rclone.conf --file-perms 0777 --dir-perms 0777 --vfs-cache-mode writes --vfs-write-back 2
%~dp0/rclone mount nspsharelan:nspshare Z: --config %~dp0/rclone.conf --file-perms 0777 --dir-perms 0777 --vfs-cache-mode writes --vfs-write-back 2 --cache-dir %~dp0/cache/nspserverlan --dir-cache-time 5 --poll-interval 4
%~dp0/rclone mount nspsharewan:nspshare X: --config rclone.conf --file-perms 0777 --dir-perms 0777 --vfs-cache-mode writes --vfs-write-back 2 --cache-dir %~dp0/cache/nspserverwan --dir-cache-time 5 --poll-interval 4 --network-mode

默认参数下不支持写入，需改为writes
--vfs-cache-mode CacheMode           Cache mode off|minimal|writes|full (default off)

修改cache同步时间：
https://rclone.org/commands/rclone_mount/#vfs-directory-cache
--dir-cache-time <seconds>   Time to cache directory entries for (default 5m0s)
--poll-interval <seconds>    Time to wait between polling for changes. Must be smaller than dir-cache-time. Only on supported remotes. Set to 0 to disable (default 1m0s)

手动刷新：
kill -SIGHUP $(pidof rclone)

文件修改后cache同步到remote的timeout：
--vfs-write-back <seconds>            Time to writeback files after last use when using cache (default 5s)

设置cache位置
--cache-dir
